# 🏥 MediAssist AI Pro

AI-powered healthcare support and triage dashboard built with React, Vite, Tailwind CSS, shadcn/ui, and Google Gemini AI.

## Features

* 🤖 Gemini AI chatbot for healthcare assistance
* 🧠 Automated intent classification and urgency detection
* 📋 Patient Support, Volunteer Registration, and Contact forms
* 📊 Analytics dashboard with healthcare metrics
* 💬 ChatGPT-style conversational interface
* 🔔 Toast notifications for action workflows
* 🌙 Modern dark SaaS UI with responsive design
* ✨ Framer Motion animations

## Tech Stack

* React + Vite
* TypeScript
* Tailwind CSS
* shadcn/ui
* Framer Motion
* Lucide Icons
* Express.js
* Google Gemini API

## Installation

```bash
npm install
```

Create a `.env` file:

```env
GEMINI_API_KEY=your_gemini_api_key
```

Run the application:

```bash
npm run dev
```
